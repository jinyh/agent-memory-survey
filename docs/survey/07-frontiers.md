# Frontiers：多模态、空间记忆与安全

> v2.0 | 2026-03-25

## 多模态记忆

- M3-Agent、MemVerse、Synapse、TeleMem 说明多模态 memory 已经不再是附属话题
- 视频、音频、视觉对象、实体锚点与时间连续性需要联合建模

## 空间记忆

- Think3D、GSMem、RenderMem 一类工作把记忆扩展到 3D 空间表征
- 重点不再是“看见了什么”，而是“环境中什么结构能被重复定位和推理”

## 安全与审计

- 长期记忆容易被投毒
- 图和多模态记忆会放大错误传播的持续时间
- 高质量 memory layer 需要来源追踪、权限边界和可撤销机制

## 对本项目的含义

- 综述主线继续以生命周期组织
- 多模态与空间记忆作为 frontier 独立保留
- `ref/DeepResearch/` 的价值是持续提供前沿摄取线索，而不是直接替代正式索引
