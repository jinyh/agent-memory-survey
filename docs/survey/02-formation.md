# Formation：记忆形成与写入

> v2.0 | 2026-03-25

## 关注问题

Formation 关注的不是“要不要存”，而是：

- 什么信息值得进入长期记忆
- 进入前如何被抽取和结构化
- 写入后以什么形式可被后续演化和检索

## 当前主要路线

### 1. 选择性抽取

- Mem0 / LangMem 路线：先从对话中抽事实，再判断是新增、更新还是忽略
- AgeMem / MemAgent 路线：把存、删、改变成 agent 学出来的动作
- 活跃态文档路线：把高时效信息先写入 always-loaded state，而不是直接打散进向量库

### 2. 结构化表示

- 原始文本与摘要块
- 向量嵌入
- 图节点与关系边
- KV 或潜在压缩表征
- 多模态片段、视频片段、空间状态

### 3. 写入边界

- 用户偏好与长期事实适合进入长期库
- 当前任务状态、短中期优先级更适合进入活跃态状态文档
- 来自工具和环境的观测要保留来源与时间戳，否则后续无法做审计和版本化
