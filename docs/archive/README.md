# Archive

这个目录用于存放已经退出主研究链路、但仍值得保留以供追溯的历史文档。

## 适合归档的内容

- 已被新版本替代的方法层文档
- 已退出主目录的流程设计、skill 设计与仓库整理文档
- 旧版 survey / survey-map / README
- 不再作为当前正式工件入口的历史材料

## 不适合归档的内容

- 当前仍在使用的正式研究工件
- 当前 survey 的正文与索引层
- 当前 memory / evaluation / references 实现相关的架构决策

## 使用原则

- 归档是为了降低主目录噪音，不是为了隐藏历史。
- 如果文档仍会被高频引用，它应留在主目录而不是 archive。
- 归档后如有 README/索引引用，应同步更新入口说明，避免失效链接。
