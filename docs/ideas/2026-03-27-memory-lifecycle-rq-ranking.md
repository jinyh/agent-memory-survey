# Agent Memory 研究问题排序与关系

## 背景

基于当前仓库已有资料，尤其是 `docs/plans/2026-03-26-memory-lifecycle-eval-research-brief.md`、`docs/survey/` 以及 `ref/paper/`、`ref/blog/` 中的证据，整理出 4 个研究问题之间的关系与优先级。

## 核心内容

这 4 个问题可以分成一条研究链：

1. **RQ-001：Agent Memory 全生命周期评测框架研究**
   - 关注“怎么衡量”
   - 是后续所有问题的共同评测地基
2. **问题 2：不同类型的记忆应如何分层检索与路由**
   - 关注“怎么找出来更有效”
   - 当前证据最成熟、最容易形成可量化结论
3. **问题 1：记忆的写入、压缩与遗忘策略，如何影响长期任务成功率**
   - 关注“记忆怎么变才是对的”
   - 需要更强的 formation / evolution 评测支撑
4. **问题 3：多模态与空间推理任务里，记忆应如何表示和更新世界状态**
   - 关注“记忆表示是否要升级”
   - 属于 frontier 方向，潜力大但当前最不稳定

### 建议排序

1. **RQ-001**：全生命周期评测框架
2. **问题 2**：分层检索与路由
3. **问题 1**：写入 / 压缩 / 遗忘策略
4. **问题 3**：多模态与空间状态记忆

### 排序理由

- **RQ-001** 是地基：先定义评测边界，才能避免后续研究只优化局部指标。
- **问题 2** 最成熟：retrieval 端证据最多，最适合先做出可验证结论。
- **问题 1** 重要但更难归因：formation 和 evolution 的实验设计比 retrieval 更复杂。
- **问题 3** 最前沿但风险最高：概念大、证据散，适合作为后续扩展方向。
